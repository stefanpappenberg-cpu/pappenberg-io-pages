FTP-Hintergruende
=================

Lege hier Hintergrundbilder ab, die du per FTP hochlaedst.

Empfehlung:
- JPG oder WebP
- Querformat
- ca. 1800 bis 2400 px Breite
- moeglichst unter 500 KB

In der Website kannst du spaeter im Hintergrund-Panel den Dateinamen eintragen,
zum Beispiel:

see-abend.jpg

Daraus wird automatisch:

images/backgrounds/see-abend.jpg
